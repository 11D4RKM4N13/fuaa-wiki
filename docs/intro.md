# Introducción

Este proyecto, desarrollado en el marco de la asignatura Ingeniería de Software II, tiene como objetivo diseñar un sistema de información para una agencia inmobiliaria con presencia en varias ciudades de Colombia, permitiendo optimizar la gestión de inmuebles, inquilinos y propietarios, así como automatizar procesos clave como la comunicación con clientes y la publicación de propiedades en un sitio web actualizado. Como estudiantes de Ingeniería de Sistemas, aplicamos nuestros conocimientos para construir una solución segura, eficiente y escalable, considerando los riesgos asociados a la protección de datos, la integridad de la información y la disponibilidad del servicio. Por ello, este documento no solo expone la propuesta técnica del sistema, sino también un análisis de los riesgos identificados y las estrategias de mitigación implementadas. A través de un enfoque de aprendizaje colaborativo, hemos distribuido tareas, investigado buenas prácticas en ciberseguridad y evaluado diversas opciones para garantizar el cumplimiento de los requisitos del cliente, asegurando un desarrollo sólido y alineado con las necesidades del proyecto.

# Objetivos

Diseñar e implementar un sistema de información seguro y eficiente para la gestión integral de alquileres de inmuebles, incorporando la automatización de comunicaciones y la sincronización web en tiempo real.

## Objetivos Específicos
1. Analizar e identificar los riesgos asociados al desarrollo y operación del sistema, estructurándolos en una matriz de riesgos para su correcta priorización.
2. Implementar medidas de seguridad tanto técnicas (como cifrado y autenticación multifactor) como organizacionales (capacitaciones y políticas de seguridad).
3. Asegurar la sincronización diaria entre la base de datos interna y el sitio web, estableciendo mecanismos de monitoreo y alertas proactivas.
4. Documentar y aplicar buenas prácticas de ciberseguridad para garantizar la protección de datos personales y financieros dentro del sistema.
